Do not hardcode addresses in your app — read them from the SDK.

```ts
import { getChainDeployment, getSolanaDeployment } from "@opaquecash/opaque";

const evm = getChainDeployment(11155111);
const sol = getSolanaDeployment("devnet");
```

Override any address via `OpaqueClient.create({ contracts: { … } })`.
