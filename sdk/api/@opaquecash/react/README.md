# @opaquecash/react

`@opaquecash/react` — React hooks for the Opaque SDK.

Wrap your tree in [OpaqueProvider](/sdk/api/@opaquecash/react/functions/OpaqueProvider.md) with a client built via
`OpaqueClient.fromWallet` (or `create`), then read it anywhere with
[useOpaqueClient](/sdk/api/@opaquecash/react/functions/useOpaqueClient.md), scan the unified inbox with [useScan](/sdk/api/@opaquecash/react/functions/useScan.md), and resolve
per-output native balances with [useStealthBalance](/sdk/api/@opaquecash/react/functions/useStealthBalance.md).
