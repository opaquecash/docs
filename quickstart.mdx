This guide takes you from zero to a working integration: install the SDK, create an `OpaqueClient`, register your meta-address, and prepare a stealth send.

## Prerequisites

- Node.js **20.17+**
- An Ethereum wallet (Sepolia testnet ETH for writes)
- Optional: Solana wallet on devnet for multichain flows
- For scan/sweep/prove: host or load the [cryptography WASM](/sdk/installation#wasm-module) module

## 1. Install

```bash
npm install @opaquecash/opaque viem
```

## 2. Derive keys from a wallet signature

Opaque derives viewing and spending keys from a **one-time wallet signature** — not from your private key directly. The message is chain-neutral:

```ts
import { OpaqueClient, SETUP_MESSAGE } from "@opaquecash/opaque";

// In the browser, prompt the user once:
const walletSignature = await walletClient.signMessage({
  account: userAddress,
  message: SETUP_MESSAGE,
});
```

`SETUP_MESSAGE` is exported from the SDK and must match [CSAP spec](https://github.com/opaquecash/spec/blob/main/CSAP.md) §2.2 byte-for-byte.

## 3. Create the client

```ts
const client = await OpaqueClient.create({
  chainId: 11155111,                              // Sepolia
  rpcUrl: "https://ethereum-sepolia.publicnode.com",
  walletSignature,
  ethereumAddress: userAddress,
  wasmModuleSpecifier: "/pkg/cryptography.js",    // required for scan/sweep/prove
  ethereumProvider: window.ethereum,              // for EVM writes (register, send, PSR)
});
```

<Note>
  PSR schema/attestation admin (`createSchema`, `issueAttestation`) does **not** require WASM. Scanning, sweeping, and proof generation do.
</Note>

## 4. Share your meta-address

```ts
const metaAddress = client.getMetaAddressHex();
// Share this 66-byte hex so senders and issuers can pay or attest to you.
```

Register on-chain so others can resolve your normal address:

```ts
// High-level — submits the tx for you
const { txHash } = await client.registerMetaAddress("ethereum");

// Or build calldata and submit via your own wallet UI
const reg = client.buildRegisterMetaAddressTransaction();
await walletClient.sendTransaction({ to: reg.to, data: reg.data, chain: sepolia });
```

## 5. Send a stealth payment

```ts
const recipient = "0xRecipientEoaAddress...";
const { metaAddressHex, registered } = await client.resolveRecipientMetaAddress(recipient);

if (!registered) {
  throw new Error("Recipient has not registered on Opaque yet");
}

// Low-level: prepare + announce calldata
const send = client.prepareStealthSend(metaAddressHex!);
const announce = client.buildAnnounceTransactionRequest(send);
await walletClient.sendTransaction({ to: send.stealthAddress, value: parseEther("0.01"), chain: sepolia });
await walletClient.sendTransaction({ to: announce.to, data: announce.data, chain: sepolia });

// High-level: one call (transfer + announce)
const result = await client.sendStealthPayment({
  chain: "ethereum",
  recipient,
  amount: parseEther("0.01"),
});
```

## 6. Receive: scan and sweep

```ts
const inbox = await client.scan({ chains: ["ethereum"] });
const balances = await client.getBalancesForOutputs(inbox);

if (inbox.length > 0) {
  const { tx } = await client.sweep({
    output: inbox[0],
    chain: inbox[0].chain,
    destination: freshAddress,
  });
}
```
